# Open in Xcode

A [Nova](https://nova.app) extension that opens the current workspace in Xcode.

## Usage

Invoke via **Extensions > Open in Xcode** or search for "Open in Xcode" in the Command Palette.

## More Info

See the internal [README](/OpenInXcode.novaextension/README.md) for more info.

Recommended download from [Nova Extensions](https://extensions.panic.com/extensions/com.gingerbeardman/com.gingerbeardman.OpenInXcode/) website.

## Requirements

- [Xcode](https://developer.apple.com/xcode/) must be installed
